# Sales Data Analysis

## Project Overview

This project analyzes sales data using SQL, Python (Pandas), Excel, and Power BI to identify business trends, KPIs, and actionable insights.

## Objectives

- Analyze sales performance
- Identify top-selling products
- Discover customer purchasing patterns
- Calculate key business metrics
- Create visual dashboards and reports

## Technologies Used

- SQL
- Oracle Database
- Python
- Pandas
- Excel
- Power BI
